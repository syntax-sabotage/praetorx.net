Scribe

Kostenlos Utility v19.0.1.1.11 OPL-1

Markdown-Unterstützung für jedes Html-Feld in Odoo. Über den Powerbox-Befehl Markdown schreiben, Live-Vorschau sehen und direkt als HTML übernehmen — mit GitHub Flavored Markdown und Code-Syntax-Highlighting.

Im Odoo App Store Alle Module

Funktionen

Powerbox-Befehl

In jedem Html-Feld "/" tippen und "Markdown" auswählen — öffnet den Scribe-Dialog. Funktioniert in allen Odoo-Modellen ohne zusätzliche Konfiguration.

Live-Vorschau

Vollständiger Wizard-Dialog mit Zwei-Panel-Ansicht: Markdown links eingeben, gerendertes HTML rechts in Echtzeit sehen. Kein Rätselraten über das Endergebnis.

GitHub Flavored Markdown

Vollständige GFM-Unterstützung: Tabellen, Aufgabenlisten mit Checkboxen, umgrenzte Code-Blöcke mit Sprachkennung und alle Standard-Formatierungen.

Code-Syntax-Highlighting

Integration mit Odoos eingebautem Prism.js für farbige Syntax-Hervorhebung in Code-Blöcken. Unterstützt Python, JavaScript, XML und viele weitere Sprachen.

Universelle Unterstützung

Funktioniert auf jedem Html-Feld in jedem Modell — Projektbeschreibungen, Aufgaben, E-Mail-Vorlagen, Wissensdatenbank-Artikel, Produktbeschreibungen.

Mehrsprachig

Benutzeroberfläche verfügbar in Deutsch, Englisch und Französisch. Keine zusätzlichen Übersetzungsschritte notwendig.

Unterstützte Markdown-Syntax

Grundformatierung

# H1 bis ###### H6 — Überschriften aller Ebenen. **fett**, *kursiv*, ~~durchgestrichen~~. [Linktext](URL) und ![Alt](Bild-URL).

Listen

Geordnete Listen (1. 2. 3.), ungeordnete Listen (- * +) und Aufgabenlisten mit - [ ] unchecked und - [x] checked.

Code

Code-Blöcke mit ```sprache für Syntax-Highlighting, Inline-Code mit `code`, Blockzitate mit > quote.

Tabellen

GitHub Flavored Markdown-Tabellen mit Spaltenausrichtung (links, zentriert, rechts) — ideal für strukturierte Vergleiche und Übersichten.
